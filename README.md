Mente Clara - Psicologia Online
Site institucional para clínica de psicologia online, focado em ansiedade, burnout e saúde mental. Desenvolvido com Lovable e hospedado no GitHub.

Sobre o projeto
O Mente Clara foi criado para resolver um problema real: pessoas que sabem que precisam de ajuda psicológica mas sentem vergonha, medo ou acham que "não é tão grave assim". O site usa uma linguagem acolhedora e sem julgamento, com design em tons suaves (sage, bege, laranja queimado) que transmite calma e profissionalismo.

Funcionalidades
Site completo com quatro páginas: Início com hero e gatilhos emocionais, Sobre com a história e valores da clínica, Blog com posts fictícios sobre ansiedade e burnout, e Contato com formulário que envia mensagem diretamente. O site também tem menu responsivo que vira hambúrguer no celular, botão flutuante do WhatsApp em todas as páginas, seção de preços transparentes com pacotes de sessões, depoimentos de clientes, e perguntas frequentes que quebram as principais objeções de quem tem receio de começar a terapia.

Tecnologias usadas
HTML5, CSS3 com arquivos separados para organização (style.css e responsive.css), JavaScript puro para o menu mobile, scroll suave e envio do formulário, e a hospedagem é feita pelo Lovable com deploy automático no GitHub.

Como executar localmente
Você pode baixar o repositório clicando em Code e depois Download ZIP, extrair os arquivos e abrir o arquivo index.html no seu navegador. Se preferir usar um servidor local para testar tudo direitinho, pode usar a extensão Live Server do VS Code.
